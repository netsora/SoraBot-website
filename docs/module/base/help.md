# 帮助

<Badge id="margin" type="warning" text="群聊"/> <Badge type="private" text="私聊"/> <Badge type="telegram" text="Telegram"/>

查询林汐内置帮助。

## 查询帮助

### 使用 `/帮助` 指令

::: tip 小贴士
若 `模块名` 位空，则发送 指令列表
:::

:::: code-group
::: code-group-item 指令

```shell 指令
@bot /帮助 [指令]
```

:::
::: code-group-item 指令别名

```shell
@bot /help [command]
```

:::
::::

### 使用 -h 子命令

使用 `-h` 或 `--help` 查询帮助

```shell
<指令> -h
```

#### 举例

* 查询帮助指令的帮助

```shell
@bot /help -h
```

* 查询 jk 指令的帮助

```shell
/jk -h
```
