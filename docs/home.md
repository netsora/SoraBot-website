# 林汐

::: info
适用版本号：v1.0.0
:::

本文档将指引您如何获取并使用 SoraBot。  
**林汐** 为 SoraBot 对外公开的昵称，本文档其它部分将使用此昵称。

## 获取机器人

你可以在以下三个平台中使用林汐：

### [QQ](./platform/qq.md)

支持在群聊、私信与频道中使用。

### [Telegram](./platform/telegram.md)

支持在群组、私信使用。
